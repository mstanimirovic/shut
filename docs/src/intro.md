# Introduction

`shut` is a minimal but well‑architected 2D top‑down shooter written in Rust using the [Macroquad](https://macroquad.rs/) game framework.  
The project serves as a learning resource for intermediate Rust developers who want to see how a real‑time game can be structured with:

- Clean separation of data and logic (systems)
- Component‑based entities
- An event queue for decoupled communication
- Unified input handling
- Simple game state machine

**Target audience**: Rust programmers with basic knowledge of ownership, structs, and enums who want to explore game development patterns.

Browse the chapters to understand every file and the reasoning behind each design decision.
